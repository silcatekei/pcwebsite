Comprehensive Operating System Installation Guide

Welcome! This guide provides step-by-step instructions for installing popular desktop operating systems. Select your desired OS below to begin.

I. Universal Preparation Steps

  1. Backup Your Data:

    Before making any significant changes like installing a new OS, ALWAYS back up your important files from existing drives to an external hard drive, cloud storage, or another computer. Data loss can occur if something goes wrong.

  2. Gather Necessary Tools & Information:
    • A USB flash drive (minimum 8GB, 16GB+ recommended). This will be erased, so ensure it doesn't contain important data.
    • Access to another working computer with an internet connection (to download the OS and create the bootable USB).
    • Product Key (if applicable): For retail versions of Windows, have your product key ready. Some OEM systems have keys embedded in firmware. Linux is generally free. macOS on custom PCs doesn't use traditional product keys in the same way.
    • Motherboard & Component Drivers: It's wise to pre-download essential drivers, especially for your Network/Wi-Fi adapter, chipset, and graphics card, from the manufacturers' websites. Store them on a separate USB or the bootable OS USB if there's space.
  3. Understand Your Hardware:

    Know your CPU (Intel/AMD), motherboard model, and graphics card. This is crucial for driver selection and troubleshooting, especially for Linux and macOS on custom PCs.

II. Installing Windows (e.g., Windows 10 / Windows 11)

A. Creating Bootable Windows USB

  1. On a working Windows PC, go to the official Microsoft website to download the Windows Media Creation Tool for your desired version (e.g., Windows 10 or Windows 11).
    • Windows 10: Download Windows 10
    • Windows 11: Download Windows 11
  2. Run the Media Creation Tool. Accept the license terms.
  3. Select "Create installation media (USB flash drive, DVD, or ISO file) for another PC" and click Next.
  4. Choose your language, edition (usually pre-selected or matches your current license), and architecture (usually 64-bit). Click Next.
  5. Select "USB flash drive" and click Next.
  6. Plug in your USB drive (ensure it's at least 8GB and has no important data). Select it from the list and click Next.
  7. The tool will download Windows and create the bootable USB drive. This may take some time.

B. BIOS/UEFI Configuration for Windows

  1. Plug the bootable Windows USB into the new PC you're building or the PC you're installing Windows on.
  2. Start the PC and immediately press the key to enter BIOS/UEFI setup. Common keys are Del, F2, F10, F12, or Esc. The key is often displayed briefly on startup or in your motherboard manual.
  3. Boot Order: Navigate to the "Boot" or "Boot Priority" section. Set the USB drive as the first boot device. Your USB drive might be listed by its name or as "UEFI USB Device."
    For modern systems, booting in UEFI mode is recommended over Legacy/CSM mode for better security and features. Ensure your USB is prepared for UEFI booting if your system supports it (Media Creation Tool usually handles this).
  4. Secure Boot: For Windows 10/11, Secure Boot should generally be **Enabled** for optimal security if your hardware supports it.
  5. SATA Mode: Ensure your SATA controller is set to **AHCI** mode (not IDE or RAID, unless you have a specific RAID setup).
  6. Save changes and exit BIOS/UEFI. The PC will restart.

C. Windows Installation Process

  1. The PC should now boot from the USB drive, and you'll see the Windows Setup screen.
  2. Select your language, time and currency format, and keyboard input method. Click Next.
  3. Click "Install now."
  4. Enter your Windows product key if prompted. If you don't have one yet or are reinstalling on a PC that was previously activated, you might be able to select "I don't have a product key."
  5. Select the edition of Windows you want to install (e.g., Windows 10/11 Home, Pro). This should match your product key.
  6. Accept the license terms and click Next.
  7. Choose the installation type: Select "Custom: Install Windows only (advanced)".
    Do NOT choose "Upgrade" if you are doing a clean install on a new PC or want to wipe an existing drive.
  8. Drive Partitioning:
    • New Drive: If it's a brand new, unformatted drive, you'll see "Drive 0 Unallocated Space" (or similar). Select it and click "New." Windows will suggest a size (usually the full size). Click "Apply." Windows will create several necessary partitions (System, MSR, Primary). Select the "Primary" partition (the largest one) and click "Next."
    • Existing Drive (Clean Install): If the drive has old partitions you want to remove, select each existing partition on the target drive one by one and click "Delete." WARNING: THIS ERASES ALL DATA ON THOSE PARTITIONS. Once you have only "Unallocated Space" on the target drive, proceed as with a new drive.
    • Formatting: If you want to install on an existing partition without deleting others (e.g., dual boot), select the desired partition and click "Format" (this erases data on that partition only), then "Next."
  9. Windows will now copy files and install. Your PC will restart several times during this process. Do not press any keys when it asks to boot from CD/DVD/USB after the first restart from the setup.
  10. Out-of-Box Experience (OOBE): After the installation, Windows will guide you through the initial setup (region, keyboard layout, network connection, account creation/sign-in, privacy settings, Cortana setup, etc.). Follow the on-screen prompts.
    For Windows 11, an internet connection and a Microsoft Account are often required during setup for Home editions. Pro editions may offer an offline account option.

II. Installing Linux (e.g., Ubuntu, Mint, Fedora)

A. Choosing a Distribution & Creating Bootable Linux USB

  1. Choose a Linux Distribution: There are many! Popular choices for beginners include:
    • Ubuntu: Very popular, large community, user-friendly. (Download Ubuntu)
    • Linux Mint: Based on Ubuntu, known for its traditional desktop and multimedia codecs out-of-the-box. (Download Linux Mint)
    • Fedora Workstation: Cutting-edge, sponsored by Red Hat, great for developers. (Download Fedora)
    Download the ".ISO" image file for your chosen distribution.
  2. Create Bootable USB:
    • Use a tool like Rufus (Windows), balenaEtcher (Windows, macOS, Linux), or the "Startup Disk Creator" (often built into Linux distributions).
    • Open the tool, select the downloaded Linux .ISO file, select your USB drive, and start the process. Ensure "Partition scheme" is set to GPT and "Target system" to UEFI if your system is modern (Rufus options).

B. BIOS/UEFI Configuration for Linux

  1. Plug the bootable Linux USB into the target PC.
  2. Enter BIOS/UEFI setup (as described in Windows section).
  3. Boot Order: Set the USB drive as the first boot device.
  4. Secure Boot: This can be tricky.
    • Some distributions like Ubuntu support Secure Boot out-of-the-box.
    • Others may require you to **Disable Secure Boot** in the BIOS/UEFI settings. If you have trouble booting, try disabling it.
  5. SATA Mode: AHCI is generally recommended.
  6. Fast Boot / Quick Boot: Some BIOS settings have "Fast Boot" options that might skip USB detection. If your USB isn't booting, try disabling Fast Boot.
  7. Save changes and exit. The PC will restart.

C. Linux Installation Process (Example: Ubuntu)

  1. The PC should boot from the USB. You'll often see a GRUB menu or a direct boot into a "live" environment.
  2. Try or Install: Most distributions offer an option to "Try Ubuntu" (or similar) without installing. This lets you test hardware compatibility. When ready, click the "Install Ubuntu" (or similar) icon on the desktop or from the initial menu.
  3. Select your language and click Continue.
  4. Select your keyboard layout and click Continue.
  5. Updates and other software:
    • Connect to Wi-Fi if possible.
    • Choose "Normal installation" (includes web browser, utilities, office software, games, media players) or "Minimal installation" (web browser and basic utilities).
    • Optionally, check "Download updates while installing Ubuntu" and "Install third-party software for graphics and Wi-Fi hardware, and additional media formats." (Recommended for most users).
    Click Continue.
  6. Installation type: This is the most critical step.
    • "Erase disk and install Ubuntu": This will wipe the entire selected hard drive and install Ubuntu as the only OS. WARNING: DELETES ALL DATA ON THE DISK. Simplest option for a dedicated Linux PC.
    • "Install Ubuntu alongside Windows Boot Manager" (or similar): If Windows is already installed, the installer might detect it and offer to resize Windows partition and install Linux in the new space (dual boot).
    • "Something else": For manual partitioning. This gives you full control to create/resize/delete partitions. Recommended for advanced users or specific dual-boot setups. You'll typically need to create at least:
      • A root partition (mount point /, ext4 filesystem, e.g., 30GB+).
      • A swap partition (optional if you have lots of RAM, typically size of RAM or double for hibernation, filesystem linux-swap).
      • (UEFI Systems) An EFI System Partition (ESP) if one doesn't exist or you're installing on a separate drive (FAT32, ~500MB, mount point /boot/efi). The installer usually handles this if you choose an automated option.
    Select your preferred option. If erasing or installing alongside, confirm the changes to the disk. Click "Install Now."
  7. If you chose manual partitioning, you'll need to create partitions and assign mount points.
  8. Select your geographical location (timezone) and click Continue.
  9. Create your user account: Enter your name, computer's name, pick a username, and choose a strong password. Click Continue.
  10. The installation will now proceed, copying files and setting up the system. This can take some time.
  11. Once complete, you'll be prompted to restart the computer. Remove the USB drive when asked.

II. Installing macOS on Custom PCs ("Hackintosh")

Important Caveat: Installing macOS on non-Apple hardware (a "Hackintosh") is a complex, unofficial process that violates Apple's End User License Agreement (EULA) for macOS. It requires specific compatible hardware, significant research, patience, and troubleshooting. Success is not guaranteed, and system stability can vary. This guide provides a very high-level overview; detailed guides like Dortania's OpenCore Install Guide are essential.

A. Pre-Requisites and Research (Crucial)

  1. Hardware Compatibility: This is paramount. Not all PC components are compatible.
    • CPU: Most modern Intel CPUs are well-supported. AMD CPUs require more work and specific patches (AMD Vanilla patches).
    • Motherboard: Chipset compatibility is key (e.g., Z390, Z490, Z590, Z690, B-series often work). BIOS settings are critical.
    • GPU: AMD Radeon GPUs (RX 500/5000/6000 series) generally have the best native support. NVIDIA support is very limited for newer macOS versions (Kepler GPUs are the last well-supported, newer ones like Turing/Ampere mostly don't work). Intel integrated graphics (iGPU) can work but may need configuration.
    • Wi-Fi/Bluetooth: Specific Broadcom chips are natively supported (often requiring a replacement M.2 card). Some Intel Wi-Fi/BT can be made to work with kexts.
    • Ethernet: Many Intel and Realtek ethernet controllers work with appropriate kexts.
    • NVMe SSDs: Most work, but some specific models can cause issues.
    Consult resources like Dortania's Anti-Hardware Buyer's Guide.
  2. Bootloader - OpenCore: OpenCore is the current standard bootloader for Hackintoshes. It's powerful but has a steep learning curve. Avoid outdated tools like UniBeast/MultiBeast.
  3. Access to a Mac or existing Hackintosh/macOS VM: You'll need this to download macOS from the App Store and create the USB installer.
  4. Patience and Willingness to Learn: This is not a plug-and-play process.

B. Creating Bootable macOS USB with OpenCore

This is a highly involved process. The Dortania OpenCore Install Guide is the definitive resource. A simplified overview:

  1. Download macOS: From the App Store on a real Mac or existing Hackintosh (e.g., macOS Monterey, Ventura, Sonoma).
  2. Format USB Drive: On macOS, use Disk Utility to format your USB drive as "Mac OS Extended (Journaled)" with a GUID Partition Map (GPT).
  3. Create macOS Installer: Use the createinstallmedia command in Terminal to make the USB bootable with the downloaded macOS installer.
  4. Gather Kexts and SSDTs:
    • Kexts (Kernel Extensions): These are like drivers. Essential ones include Lilu, VirtualSMC, WhateverGreen (for graphics), AppleALC (for audio), and kexts for your Ethernet/Wi-Fi.
    • SSDTs (System Description Table): These are ACPI patches to make your hardware appear more Mac-like. You'll often need pre-built ones or learn to create custom ones (e.g., SSDT-EC, SSDT-PLUG, SSDT-AWAC/RTC).
  5. Configure OpenCore (`config.plist`): This is the core of your Hackintosh setup. You'll create an EFI folder on your USB, place OpenCore files, kexts, SSDTs, and ACPI drivers. The `config.plist` file needs to be meticulously configured for your specific hardware (CPU generation, chipset, devices, quirks, etc.). Tools like ProperTree are used to edit it. Dortania's guide walks through each section.
  6. Copy the configured EFI folder to the EFI partition of your USB installer.

C. BIOS/UEFI Configuration for macOS (Hackintosh)

These settings are critical and vary by motherboard. General guidelines (refer to Dortania's guide for specifics):

  • Disable: Fast Boot, Secure Boot, Serial/COM Port, Parallel Port, VT-d (can be enabled if DisableIoMapper quirk is set), CSM (Compatibility Support Module), Thunderbolt (unless specifically configuring it), Intel SGX, Intel Platform Trust.
  • Enable: VT-x, Above 4G decoding, Hyper-Threading, Execute Disable Bit, EHCI/XHCI Hand-off, OS type: Windows 8.1/10 UEFI Mode or Other OS, DVMT Pre-Allocated (iGPU Memory): 64MB or higher.
  • SATA Mode: AHCI.

Set USB drive as the first boot device. Save and exit.

D. macOS Installation Process (Hackintosh)

  1. Boot from the OpenCore USB installer. You should see the OpenCore boot picker. Select the macOS installer.
  2. If it boots successfully into the macOS Utilities screen (verbose text may scroll by first):
    • Open Disk Utility.
    • Select your target SSD/HDD for macOS. Click "Erase." Name it (e.g., "Macintosh HD"), choose **APFS** as the format, and "GUID Partition Map" as the scheme. Click Erase. Close Disk Utility.
  3. Select "Install macOS [Version Name]" and click Continue.
  4. Follow the on-screen prompts, agreeing to the license and selecting your newly formatted APFS drive as the destination.
  5. The first stage of installation will copy files and then restart. The PC should automatically boot from the USB again, and OpenCore should now show an option like "macOS Installer" or the name of your drive. Select it to continue installation. This may happen a few times.
  6. Eventually, it will boot into the macOS welcome screen/setup assistant.
  7. Post-Install:
    • Copy the EFI folder from your USB's EFI partition to the EFI partition of your main macOS drive. This makes your system bootable without the USB.
    • Install necessary kexts not already in your EFI, configure audio, graphics acceleration, USB mapping, Wi-Fi/Bluetooth, iServices (iMessage, FaceTime - requires careful SMBIOS configuration), power management, etc. This is a significant part of the Hackintosh process.
    • Refer extensively to Dortania's Post-Install guide.
Troubleshooting is common. If you get stuck, search forums like r/hackintosh, InsanelyMac, or TonyMacx86 (though the latter focuses more on their tools) with detailed information about your hardware and the issue.

III. Universal Post-Installation Steps

  1. Install Drivers:
    • Windows: Windows Update usually finds many drivers. However, always install the latest chipset drivers from your motherboard manufacturer and graphics drivers from NVIDIA/AMD/Intel directly. Check Device Manager for any devices with yellow exclamation marks.
    • Linux: Most hardware works out-of-the-box. For NVIDIA graphics, you might want to install proprietary drivers via the "Additional Drivers" tool (Ubuntu/Mint) or package manager. For some Wi-Fi cards, you might need to find specific firmware or drivers.
    • macOS (Hackintosh): This is an ongoing process of ensuring all kexts are correct and hardware is properly configured (as mentioned in its post-install section).
  2. Update the Operating System:
    • Windows: Run Windows Update repeatedly until no more important updates are found.
    • Linux: Open a terminal and run commands like sudo apt update && sudo apt upgrade -y (for Debian/Ubuntu-based) or sudo dnf upgrade -y (for Fedora).
    • macOS: Check for Software Updates in System Settings/Preferences. Be cautious with major OS updates on a Hackintosh; always check community compatibility first.
  3. Install Essential Software: Web browsers (Chrome, Firefox), office suite (LibreOffice, Microsoft Office), media players (VLC), security software (if desired), and any specific applications you need.
  4. Customize Settings: Adjust display resolution, personalize your desktop, set power options, configure privacy settings, etc.
  5. Test Everything: Ensure audio, video, networking, USB ports, and peripherals are working correctly.
  6. (Optional) Restore Data: If you backed up data earlier, restore your personal files.
  7. Remove Bootable USB: You can now remove the installation USB. If needed, go back into BIOS/UEFI and set your main OS drive (e.g., "Windows Boot Manager" or your Linux/macOS drive) as the primary boot device.

Congratulations! Your new operating system should be ready to use.

Disclaimer: This is a general guide. Specific steps, prompts, and interfaces may vary based on your hardware, chosen OS version, and distribution. Always refer to official documentation and reputable community guides if unsure or for advanced configurations.
